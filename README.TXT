How to run:

use the following command in the directory to run the generator:

make -f generator.mak 

Use the following command in the directory to run the dumper:

make -f dump.mak

Notes: 
Want to edit the logged messages?
- change line 3 in generator.mak: ./generator iwork1 iwork2 iwork3 iwork4 iwork5 iwork6 iwork7 iwork8
edit any of iwork1 to iwork8 to contain a new message to store.

SQLite must be setup and properly installed for code to compile.
